Features

核心功能

每一个细节都经过精心打磨，只为给你最纯粹的音乐体验

Rust 原生引擎

基于 symphonia + cpal 构建的高性能音频解码引擎，低延迟、低资源占用，支持 MP3、FLAC、WAV、OGG、AAC 等主流格式。

多平台音源

整合网易云音乐、酷我音乐、酷狗音乐三大平台，海量曲库一键搜索，想听什么都能找到。

智能缓存与下载

LRU 磁盘缓存加速二次播放，支持高品质下载并自动写入 ID3 标签、封面和歌词文件。

双主题切换

精致的深色毛玻璃拟物风和明亮的天蓝浅色模式，白天黑夜都有最佳视觉体验。

收藏与歌单

自由收藏管理，拖拽排序播放列表，发现推荐歌单和排行榜，打造个人专属曲库。

本地音乐管理

导入本地音频文件夹，自动读取元数据和封面，离线也能畅享你的音乐收藏。

Installation

安装指南

简单几步，即刻开始你的音乐之旅

1

下载安装包

点击页面顶部的「下载安装包」按钮，或前往 下载页面 获取最新版本的 Windows 安装包（.msi 或 .exe）。

2

运行安装程序

双击下载的安装文件，按照提示完成安装。默认安装路径为 C:\Program Files\Murmble，你也可以自定义安装位置。

3

启动音源服务

Murmble 依赖本地音源服务获取曲库。进入安装目录中的 unified-api 文件夹，运行 unified-api.exe，确保服务在 localhost:8080 正常监听。

4

开始使用

启动 Murmble 应用，搜索你喜欢的音乐，享受纯净无广告的高品质聆听体验吧！支持多种播放模式：顺序、随机、单曲循环。
